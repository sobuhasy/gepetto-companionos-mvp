# Gepetto CompanionOS v0.1 — Signal Demo

## Goal

Build a visible, privacy-first AI companion workspace that proves the pitch deck story:

From chatbot to CompanionOS.

First the signal. Then the face. Then the vessel.

## Generated Identity

The demo uses a deterministic local generated companion identity engine. The user sees one persistent generated companion profile, not a list of fixed personas.

Modes change behavior, not identity. The same companion can study, code, plan, coach Japanese, brainstorm, or offer non-clinical emotional support.

Aerilonian and Dimension-7-Lyra are product lore flavor. The companion should stay understandable, privacy-first, and should not claim literal real-world sentience.

## Core Demo Loop

1. Open Gepetto CompanionOS.
2. See one persistent companion identity.
3. Choose a mode: Study, Code, Productivity, Japanese Coach, Creative, or Emotional Support.
4. Run a task or conversation.
5. Save memory or complete a mission.
6. Show privacy/consent controls.
7. Explain roadmap: Signal now, Face later, Vessel later.
